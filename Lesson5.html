Applying CSS Selectors Specificity Colors
  1. Inline: In the start of an element
    2. <div style="margin:10px;"...>

  2. Embedded: In css is in the head tag
  3. External: imported in the head tag
  1. Element selector: div, table, main - element names
    2. div { margin-top: 10px; }

  2. Id selector using # (in the start tag)
    3. #td2 { font-size: 2em; }

  3. Class selector using .
    4. .divClass { color: green; }

  4. Descendent selector
    5. div .liStyle { color: green; }

  5. Pseudo class selector
    6. :hover { color: blue; }
  6. Attribute selector
    7. a[target=_blank] { color: red; text-size: 2em;}
  • Inline style take precedence over embedded styles
  • Embedded style take precedence over css
  • EXCEPTION !important keyword
  • RGB (red, green, blud)
    • values 0 - 255, 0 is black, 255 is white (highest intensity)
  • hexidecimal
    • start with #, 6 digit alphanumeric value
    0-9a-f - 16 possible values
    #000000 is black
    - #000; shortcuts #ffffff is white
    - #fff;
  • Color names
    • 140 names, most IDES will show in intellisense

Box Model Display Float and Clear Font and Text
Content: text and images
Padding: transparent whitespace around the content
Border: contains the padding and the content
Margin: transparent whitespace around the border
 Display controls elements layout.
inline - displays as an inline element like span or a
block - displays as a block element like p, table, or div
inline-block displays
as block inline with text
Float specified have elements align within their parent
  • left - parent's left side
  • right - parent's right side
  • none - default
Clear stops elements from floating
  • both - no elements will float
  • clear left - no elements float to parent's left
  • clear right- no elements float to parent's right
